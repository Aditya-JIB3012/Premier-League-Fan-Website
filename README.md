# Premier League Fan Website ⚽

A responsive multi-page fan website dedicated to the English Premier League.

## Features
- Responsive navigation bar with dropdown menu
- Multi-page layout (Home & Fantasy sections)
- Interactive UI using JavaScript
- Embedded videos and media content
- Mobile-friendly and responsive design

## Tech Stack
- HTML
- CSS
- JavaScript

## Status
Ongoing – continuously improving UI and adding features.
